# TrustPay Smart Contracts

Solidity smart contracts for TrustPay escrow platform.

## Quick Start

```bash
# Install dependencies
npm install

# Compile contracts
npm run compile

# Run tests
npm test

# Deploy to Mumbai testnet
npm run deploy:mumbai
```

## Setup

1. Copy `.env.example` to `.env`
2. Add your private key
3. Get test MATIC from https://faucet.polygon.technology/
4. Deploy!

## Contract Address

After deployment, the contract address will be saved in `deployments/mumbai.json`
